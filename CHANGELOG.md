# Changelog

## 0.1.0 (2025-12-14)

### Added

- Initial release of pdfwm
- Text-based watermark generation with automatic font sizing
- Support for RGB and RGBA hex color codes
- Single file and batch processing modes
- Glob pattern support for file selection
- Recursive directory processing with structure preservation
- Overwrite mode with temporary file safety
- Custom filename suffix support
- Comprehensive error handling and validation
- Type hints for better IDE support
- CLI tool with `pdfwm` command
- Python library API for programmatic use

### Dependencies

- PyPDF2 >= 3.0.0
- reportlab >= 4.0.0
